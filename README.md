# APLICATIVO WEB RUTAPP – Gestión de Rutas Escolares

## Descripción

RUTAPP es una aplicación web desarrollada en Python con Flask para la gestión de rutas escolares.
Permite administrar conductores, estudiantes, vehículos y rutas, facilitando la comunicación entre administradores, conductores y padres de familia.

## Tecnologías utilizadas

* Python
* Flask
* MySQL
* HTML
* CSS
* JavaScript

## Estructura del proyecto

RUTAPP
├── basedatos (scripts SQL)
├── maquetacion (diseño inicial HTML)
├── rutapp (aplicación Flask)
│   ├── routes
│   ├── static
│   ├── templates
│   └── app.py
├── .gitignore
├── README.md
└── requirements.txt

## Instalación del proyecto

1. Clonar el repositorio

git clone https://github.com/cisabelsalazar/rutapp.git

2. Crear entorno virtual

python -m venv venv

3. Activar entorno virtual

Windows:
venv\Scripts\activate

4. Instalar dependencias

pip install -r requirements.txt

## Ejecución

Ejecutar el archivo principal:

python app.py

Luego abrir el navegador en:

http://localhost:5000
